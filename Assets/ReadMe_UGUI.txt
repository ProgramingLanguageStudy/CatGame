1. UGUI란?
Unity Graphical User Interface
유니티에서 제공하는 GUI
GUI: 유저가 아이콘, 버튼, 메뉴 등의 그래픽 요소를 통해
프로그램과 상호작용할 수 있도록 하는 시각적 인터페이스

2. 캔버스(Canvas)
캔버스는 UI 요소를 화면에 보여 주는 컨테이너 역할
모든 UI 요소는 캔버스의 하위에(자식오브젝트) 있어야 한다.

1) Canvas
이 게임오브젝트가 UI 요소들의 부모인 캔버스 역할을 하도록 하는 컴포넌트

2) Canvas Scaler
캔버스가 각 기기의 화면 크기에 따라 어떻게 조정될지를 설정하는 컴포넌트
